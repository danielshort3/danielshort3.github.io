## Class identity concept

Create a production concept board for three adult Fracture Runner class families. Preserve the approved 5.5-6-head proportions, realistic face and eyes, compact athletic build, charcoal/navy frontier-salvage materials, weathered iron, and restrained fracture-crystal hardware. Show each at equal scale in idle and run poses on a neutral charcoal board.

- Mage: narrow upright silhouette, fitted jacket with split utility-mantle panels above the knees, relay harness, compact forearm and upper-back conduits, cyan-violet accents.
