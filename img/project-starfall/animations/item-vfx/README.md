# Project Starfall Item VFX

Generated generic item VFX sheets live here. They are produced by:

```bash
npm run build:project-starfall-item-visual-assets
```

Current runtime policy: item pickups use existing loot fly effects and rarity aura rendering. These generated sheets are available for future renderer/data references.

Generated files include:

```text
vfx-rarity-common-drop.png
vfx-rarity-uncommon-drop.png
vfx-rarity-rare-drop.png
vfx-rarity-epic-drop.png
vfx-rarity-relic-drop.png
vfx-currency-sparkle.png
vfx-quest-pulse.png
vfx-boss-drop-reveal.png
```

VFX should support the item. It must not hide the item silhouette.
