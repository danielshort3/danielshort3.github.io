# Project Starfall Pickup Source Sheets

This folder is for future pickup source sheets and generated working files.

Use this only after the runtime has a dedicated pickup asset field. Until then, inventory icons remain the source for world loot visuals.

Future source sheet rules:

- Equal-size cells.
- `#00ffff` guide lines if processed by a grid script.
- Transparent or chroma-key background according to the processor.
- One pickup per cell.
- Stable scale and anchor point.
- No labels, no fake UI, no watermarks.
